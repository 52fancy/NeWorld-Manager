## NeWorld Manager

基于 WHMCS 下 Licensing Module 的一套自动模块部署工具

##### 简介

当前模块包含了许多 WHMCS 扩展所涉及的特性与文件操作示例，并且、这是一个老旧的项目

由于代码风格迥异且可能存在未知的安全隐患，因此将暂停维护、可提供 WHMCS 扩展开发新手用于学习
